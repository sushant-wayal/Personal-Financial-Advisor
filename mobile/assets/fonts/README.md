Place the Material Symbols Outlined TTF in this folder and name it `MaterialSymbolsOutlined.ttf`.

How to get the font:
1. Visit https://fonts.google.com/icons?icon.platform=web and search for "Material Symbols Outlined".
2. Download the font (choose the Outlined style) and extract the `.ttf` file.
3. Move the `.ttf` to `mobile/assets/fonts/MaterialSymbolsOutlined.ttf`.

After placing the file, restart the Expo app (`npm run android` or `expo start`) — the app will load the font at startup and icon glyphs should render exactly like the design.
